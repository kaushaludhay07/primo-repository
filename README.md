DOMANDE:
E UNA TAG CHE SI SIMBOLEGGIA IL FATTO CHE STIAMO PARLANDO DI UN PAGINE WEB.
IL TAG H1 CHE T FA INSERIRE UN TESTO CHE E IN GRASETTO, A LIVELLO DI GRANDEZZA DI H2 CHE SARA A SUA VOLTA PIU GRANDE DI H3.
IL TAG <p> SERVE PER CREARE UN PARAGRAFO.
    LA RIGA 
   LA RIGA  <a href="/me.html">MY OTHER PAGE </a>SERVE PER FARE IL MODO CHE QUANDO SI CLICCA SUL TESTO "MY OTHER PAGE " TI RIMANDERA ALLA PAGINA HTML.
   PERCHE NEL FILE style.css ABBIAMO DETO CHE IL COLORE DI BACKGROUND DEL BODY SERVE ESSERE ARANCIONE.
   IL TAG <link rel="stylesheet" href ="style.css"> SERVE PER COLLEGARE UNA FILE CSS
   IL SELETTORE IN UNA FILE CSS SERVE PER SCEGLIERE A QUALI ELEMENTI HTML APPLICARE 
   PER FARE COMPARE IL TESTO"HELLO" SULLA CONSOLA BISOGNA INSERIRE IL TESTO NEL FILE JAVA SCRIPT SCRIVENDDO consol.log()
   PER SCRIVERE SULLA CONSOLA CIAO AL POSTO DI  HELLO HO INSERITO "CIAO POSTO DI " HELLO"NELLE VIRGOLEETE DELLA PARENTESI.
   SI PUO VEDERE LA SCRITTA "CIAO" NELLA CONSOLA 
   COMPARE UN POPUP CON LA SCRITTA "CIAO"
   la DIFFRENZA TRA COLSOL.LOG() E ALERT() POSIZIONE IN CUI VIENE VISAULIZZZATO OUTPUT E IL MODO IN CUI INFLUISCE SULL ESECUZIONE DEL CODICE.
